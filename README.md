# Frontend JavaScript & TypeScript ALX Project

## Overview
This project is part of the ALX Software Engineering curriculum.  
It helps learners practice **TypeScript**, **interfaces**, **classes**, **DOM manipulation**, **namespaces**, and **advanced TypeScript features** through small tasks.

## Tasks Implemented

### Task 0: Creating an interface for a Student
- Created a `Student` interface with `firstName`, `lastName`, `age`, and `location`.
- Created two student objects and an array `studentsList`.
- Rendered a table in the browser showing each student’s first name and location.

**Files:**  
- `0x04-TypeScript/task_0/js/main.ts`  
- `0x04-TypeScript/task_0/package.json`  
- `0x04-TypeScript/task_0/.eslintrc.js`  
- `0x04-TypeScript/task_0/tsconfig.json`  
- `0x04-TypeScript/task_0/webpack.config.js`

### Other Tasks
- Task 1–5 implement Teachers, Directors, classes, namespaces, declaration merging, and nominal typing.
- Each task is located in its respective directory (`task_1`, `task_2`, etc.) with proper TypeScript configuration files.
